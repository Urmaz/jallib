keyboard

keyboard, scans 4x4 keybaord and returns 0x00..0x0F is keypressed, 0x10 if none pressed

Author Javier Martínez, Copyright (c) 2003, all rights reserved.
Adapted-by Eur van Andel, eur@fiwihex.nl
Compiler >=2.4g

Description 

This library consist of a single funtion: getkey() that scans a 4x4 matrix keyboard. 
it needs two variables: keydrive and keyscan.
