Description

Device include file for pic16f88, containing:

- Declaration of ports and pins of the chip.- Procedures for shadowing of ports and pinsto circumvent the read-modify-write problem.- Symbolic definitions for config bits (fuses)- Some device dependent procedures for commonoperations, like:. enable-digital-io.
