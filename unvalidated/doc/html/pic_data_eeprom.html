pic_data_eeprom

reading/writing of PIC's data EEPROM

Author Stef Mientki, Copyright (c) 2002..2006, all rights reserved.
Adapted-by Sebastien Lelong, Rob Hamerling.
Compiler >=2.4g

Description 

Write/read 8-bits data to/from data-eeprom


Notes 

Limited to the first 256 bytes of EEPROM memory


Dependencies

No dependency found

API details

Global variables/contants

Procedures

  • data_eeprom_read_word(byte in address, word out data) 

    Read word from data eeprom at given address
Result is stored in the passed in-argument

  • data_eeprom_read_dword(byte in address, dword out data) 

    Read a dword from data eeprom at given address
Result is stored in the passed in-argument

  • data_eeprom_read(byte in address, byte out data) 

    Read byte from data eeprom at given address
Result is stored in the passed in-argument

  • data_eeprom_write(byte in address, byte in data) 

    Write byte to data eeprom at given address
Wait until finished

  • data_eeprom_write_word(byte in address, word in data) 

    Write word to data eeprom at given address

  • data_eeprom_write_dword(byte in address, dword in data) 

    Write dword to data eeprom at given address

Functions

  • data_eeprom_word(byte in address) return word 

    Read word from data eeprom at given address and return result

  • data_eeprom(byte in address) return byte 

    Read byte from data eeprom at given address and return result

  • data_eeprom_dword(byte in address) return dword 

    Read dword from data eeprom at given address and return result
