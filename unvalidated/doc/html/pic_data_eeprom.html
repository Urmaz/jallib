pic_data_eeprom

reading/writing of PIC's data EEPROM

Author Stef Mientki, Copyright (c) 2002..2006, all rights reserved.
Adapted-by Sebastien Lelong.
Compiler >=2.4g

Description

this library can write/read 8-bits data to/from data-eeprom

API details

Global variables/contants

Procedures

  • data_eeprom_read( byte in address, byte out data )

    Reads a byte from data eeprom at given address
    Result is stored in the passed in argument

  • data_eeprom_read_dword( byte in address, dword out data )

    Reads a dword from data eeprom at given address
    Result is stored in the passed in argument

  • data_eeprom_write(byte in address, byte in data)

    Writes a byte to data eeprom at given address
    Waits untill finished

  • data_eeprom_write_word(byte in address, word in data)

    Writes a word to data eeprom at given address
    (based on procedure data_eeprom_write)

  • data_eeprom_write_dword(byte in address, dword in data)

    Writes a dword to data eeprom at given address
    (based on procedure data_eeprom_write)

  • data_eeprom_read_word( byte in address, word out data )

    Reads a word from data eeprom at given address
    Result is stored in the passed in argument

Functions

  • data_eeprom_word(byte in address) return word

    Reads a word from data eeprom at given address
    and return the result

  • data_eeprom(byte in address) return byte

    Reads a byte from data eeprom at given address
    and return the result

  • data_eeprom_dword(byte in address) return dword

    Reads a dword from data eeprom at given address
    and return the result
